  • LaTeX is a markup language.
  • Formats plain text into a styled document.
  • Allows you to focus on the content of your document by taking care of the formatting for you. This is unlike a "What You See Is What You Get" editor, where the user has to directly format the entire document.
  • LaTeX is especially useful for documents which require heavy use of special characters.

Basic Syntax

\commandname[option0,option1,...]{argument0}{argument1}...
\textbf{bolded text} this part is normal

\switchname scope
{\bfseries bolded text} this part is normal
  • A command takes a comma-delineated list of options, which change the behavior of the command, and a list of arguments, each surrounded by {}. Options are always optional, while arguments are not.
  • Switches act a bit differently. Switches take no arguments, but rather apply to the rest of the scope, which you can limit by surrounding it with {}. If not limited, the switch will take the rest of the document as the scope.

Example Document

\documentclass[12pt]{article} % This defines the document class, takes options and an argument

\title{My First Doc}
\author{Michael Cheng, Tina Chen} % Preamble contains commands which affect entire document
\date{\today}

\begin{document}        % Tells LaTeX the document is beginning
\maketitle
Hello World!                       This is a \LaTeX\ document % Multiple whitespace is ignored
\end{document}          % Tells LaTeX the document has finished
  • Every document starts with the \documentclass command. This command sets the base formatting style for the entire document. In this case, the argument given to the command is article, so LaTeX renders the document in the style of an article. It also takes font size as an option.
    • This is one of the strongest features of LaTeX, as it allows basic document templates to be created. Organizations like journals, universities, or companies might create their own document class so that all documents they publish will be in the same format. Unlike an editor like Microsoft Word, where each document has to be manually formatted to fit the specifications, you can simply change the document class in LaTeX.
  • The \begin command marks the beginning of a new environment. This tells LaTeX to render everything between the \begin and \end commands in the given document environment.
  • The \maketitle command creates the title of the document using information given in the preamble.
